<h2 align="center"> Quark，构建下一代前端组件！ </h2>  

### 介绍  

Quark 是一个拥有完美开发体验的 web components 框架。


### 特性

* 可以在任何框架或者无框架使用 QuarkC 自定义元素
* 小巧的尺寸和高性能设计
* Web Components + TSX 融合
